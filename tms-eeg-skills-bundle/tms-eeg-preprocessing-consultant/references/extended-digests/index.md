## Purpose

Use this folder for manually prepared long-form article digests. These files are loaded only when compact paper cards are not enough.

## Naming Rule

Use the same ID as the compact paper card, with `.extended.md` suffix.

Example:

```text
papers/wu-2018-artist.md
extended-digests/wu-2018-artist.extended.md
```
