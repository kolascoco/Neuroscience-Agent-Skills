## Role In The Skill

Use NaviNIBS as open-source neuronavigation context for registration, tracking, target planning, and reproducible coil positioning.

## Planning Use

Relevant when the user needs an open-source navigation workflow, repeated-session coil positioning, electrode digitization context, or target/coil-pose logging.

## Failure Modes

Assuming navigation alone guarantees physiological target engagement; failing to record registration error, coil deviation, and target drift; not combining navigation with online TEP QC.
