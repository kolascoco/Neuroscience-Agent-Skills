## Target Guide

| Target | Load | Main risks |
|---|---|---|
| M1 | `targets/m1.md` | MEP/RMT overreliance, i-TEP muscle contamination, coil orientation dependence |
| DLPFC | `targets/dlpfc.md` | craniofacial muscle, auditory/somatosensory confounds, weak early local TEP |
| PPC / parietal | `targets/ppc.md` | source/topography interpretation, target precision, i-TEP window choice |
| S1 / somatosensory / tactile feeling | `targets/somatosensory-s1.md` | scalp sensation versus cortical tactile processing, sensory PEPs, subjective-report confounds |
| Occipital | `targets/occipital.md` | phosphene/visual state effects, natural frequency interpretation |
| SMA / medial motor | `targets/sma.md` | target subregion specificity, connectivity/excitability distinction |
| language/lateral frontal | `targets/dlpfc.md` plus target-specific notes | muscle contamination, discomfort, sham matching |

## Rule

If the requested target is not listed, route by nearest anatomy and ask for artifact expectations, target coordinates, and backup target.
